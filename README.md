# Data Analytics Project 3

## DecodeLabs Data Analytics Internship

### Project Objective
Perform SQL-based data analysis on a retail sales dataset.

### SQL Analysis Performed
- View all records
- Delivered Orders Analysis
- Orders by Payment Method
- Orders by Referral Source
- Top Selling Products

### Key Insights
- Total records analyzed: 1200
- Multiple payment methods were used by customers.
- Referral channels contributed significantly to customer acquisition.
- Product demand varied across categories.
- SQL queries were used to extract business insights efficiently.
